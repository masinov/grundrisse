# Author Naming Convention

## General Rules

1. **Full given name + surname** (prefer full names over initials)
   - ✓ "Vladimir Lenin"
   - ✗ "V. I. Lenin", "Lenin"

2. **Use commonly known form** (not always legal name)
   - ✓ "Joseph Stalin" (not "Joseph Vissarionovich Dzhugashvili")
   - ✓ "Leon Trotsky" (not "Lev Davidovich Bronstein")
   - ✓ "Mao Zedong" (modern pinyin, not "Mao Tse-tung")

3. **Initials only when commonly known that way**
   - ✓ "C.L.R. James" (always known by initials)
   - ✓ "H.G. Wells"
   - ✗ "V. I. Lenin" → use "Vladimir Lenin"

4. **Western name order** (given name + surname)
   - ✓ "Mao Zedong" (not "Zedong Mao")
   - ✓ "Ho Chi Minh"

5. **Standard romanization for non-Latin scripts**
   - Modern pinyin for Chinese: "Mao Zedong" not "Mao Tse-tung"
   - Standard transliteration for Russian, Arabic, etc.

6. **Diacritics included when standard**
   - ✓ "José Martí"
   - ✓ "León Trotsky" (if Spanish context)

## Examples

| Variants Found | Canonical Form | Reason |
|----------------|----------------|---------|
| Vladimir Lenin, V. I. Lenin, V.I. Lenin, Lenin | **Vladimir Lenin** | Full name preferred |
| Joseph Stalin, J. V. Stalin | **Joseph Stalin** | Full name preferred |
| Friedrich Engels, Frederick Engels | **Friedrich Engels** | German spelling standard |
| Mao Tse-tung, Mao Zedong | **Mao Zedong** | Modern pinyin |
| C.L.R. James, C. L. R. James | **C.L.R. James** | Commonly known by initials, no spaces |
| Dora B. Montefiore, Dora Montefiore | **Dora B. Montefiore** | Include middle initial if used |
| Anton Pannekoek, Antonie Pannekoek | **Anton Pannekoek** | Common Dutch form |
